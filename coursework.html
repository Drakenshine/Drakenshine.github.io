Introduction to Web Design

PROG109

Description: PROG109 introduces fundamental programming and mark-up techniques for the Web. Students design procedures and write computer instructions to solve business related problems with a focus on simple applications and games. Students learn procedural programming, develop user interfaces, and work with events and objects. The class will teach client-side web development fundamentals.

5 Credits

Instructor: Maricel Medina-mora

I had very little HTML and CSS experience (Zero Javascript) before the class. I feel like this class was pretty easy to pick up as I took a C# class - getting your feet wet into a coding language definitely makes learning other languages easier.
